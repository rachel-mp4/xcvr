# xcvr

xcvr (transceiver) is a web client for [lrc](https://moth11.net/blog/lrc) built with svelte

see it in action at [xcvr.chat](http://xcvr.chat/)

## build instructions
```
npm install
npm run dev
```

Note that this is just a client. For an lrc server, use [lrcd](https://github.com/rachel-mp4/lrc)
